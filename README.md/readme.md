# 🏥 Hospital Management System

A complete Hospital Management System built with Java OOP, MySQL database, and HTML/CSS/JS frontend.

## Features

- ✅ **Patient Management** - Add, view, update, delete patients
- ✅ **Doctor Management** - Manage doctor information
- ✅ **Appointment Scheduling** - Book and track appointments
- ✅ **Billing System** - Track payments and pending bills
- ✅ **Room Management** - Manage room availability
- ✅ **Admission Tracking** - Track patient admissions
- ✅ **Medicine Inventory** - Manage medicines stock
- ✅ **Prescriptions** - Issue and track prescriptions

## Technologies Used

- **Backend:** Java (OOP concepts: Inheritance, Polymorphism, Encapsulation, Abstraction)
- **Database:** MySQL
- **Frontend:** HTML, CSS, JavaScript
- **API:** Java HTTP Server (RESTful endpoints)
- **JSON Processing:** Gson library

## OOP Principles Demonstrated

- **Inheritance:** `Patient` and `Doctor` classes extend abstract `Person` class
- **Polymorphism:** `getRole()` method returns different values based on object type
- **Encapsulation:** Private fields with public getters/setters
- **Abstraction:** Abstract `Person` class and `DBConnection` class hide implementation details
